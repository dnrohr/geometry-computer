# Getting started

Enter an explicit expression such as `(3*a + b) * (a + b)`, set sample values, and choose **Compile construction**. Select a generated step to reveal its geometry, click any SVG object to inspect its role and dependencies, and choose **Why?** to open its proof card. Gallery buttons replace both expression and sample values.

The reveal slider provides deterministic construction time. Scaffolding can show all auxiliary geometry, only the current step, or a clean result. Use Alt+Up/Down to navigate steps; all diagram objects, proof claims, and controls are keyboard focusable.
