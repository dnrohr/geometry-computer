# Accessibility

Interactive geometry exposes button roles, names, focus rings, Enter/Space activation, and persistent click selection. Steps and proof claims are native buttons. Alt+Up/Down changes the active step. Inputs, export controls, the reveal range, and scaffolding select have accessible labels.

`prefers-reduced-motion` disables transitions and animation dependence. The reveal slider and step buttons provide non-scroll alternatives. Color is reinforced with width, dash, labels, active borders, and text roles.
